# 安装方式
目前，安装方式有以下几种：


## 方式1：通过 npm 安装
```
npm install web-tls
```




## 方式3：通过`<script>`标签引入
您可直接从项目的 [发行地址][] 中下载以 `.iife.js` 作为缀的文件，然后使用如下代码引用 和 使用 web-tls：


1. 引用 web-tls
   ```
   <script src="path/to/package/web-tls.iife.js"></script>
   ```
   
2. 使用全局的 `WebTls`
   ```
   <script>
   // 使用全局的 WebTls
   </script>
   ```
